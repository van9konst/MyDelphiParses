Из папки не удалять никакие файлы вообще!
Все файлы, нужны для успешной работы софта

Перед тобой запущен софт. В нём есть 4 поля и 3 кнопки. 

Поля -
Search Request - сюда ты вводишь сам запрос для парсинга. Например : Sushi, Cinema, Theatre, Pizza, etc
CLIETNT_ID - строка с критически важными данными для успешного запроса. Не трогать.
CLIETNT_SECRET - строка с критически важными данными для успешного запроса. Не трогать.
Location - город для поиска. Вводить только в том формате, который стоит по дефолту.

Кнопки
Значит очерёдность работы должна быть следующей
1. Проверить введеные данные. Заполены ли все поля.
2. Нажать Search. Если всё хорошо, то в левом Lines будет некое число.
3. Нажать Parse. Если всё хорошо, то в правом Lines будет некое число.
4. Нажать Replace. Подождать пока софт проведён нужные операции. Если всё хорошо, то в папке с софтом
появится заполненый инфой файл DONE.
